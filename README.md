### Tadam CPP - Расширение для Visual Studio Code
## Описание
Добрый день, коллега. Перед тобой плагин-расширение для VS Code, вежливо приветствует пользователя и мотивирует его на активную и продуктивную работу.

## Установка

1. Откройте VSCode
2. Перейдите в панель расширений
3. Найдите "Tadam CPP"
4. Нажмите "Install"

### Возможности
 - приветствует пользователя
 - заряжает мотивационной речью

 ## Использование
 Запустить VS Code

### Требования
 Visual Studio Code версии 1.103.0 или выше
